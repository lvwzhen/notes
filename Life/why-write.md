# 为什么写？

距离最后一次写博客已经有一年多了，自从两年前一时冲动买域名写博客到现在足足已经过去了两年。我常常问自己这两年来有哪些变化，写博客居然是这变化中的始终。自学网站、折腾 Ubuntu、组建工作室，这两年的种种变化都和博客有关系。

每当我写博客时，总是最安静的时刻，总是最思考的时刻。现在断了一年的博客，也断了大半年的日记，时常会有心中空虚无聊的感觉。没思路时，即使是回看电影和聆听音乐也无法提起自己的兴趣。我努力回想自己的症状，发现唯有以前写博客才是心中最平静的时候。当键盘敲出这些胡言乱语时，我似乎是在和自己在说话了，这一刻彻底的放松了。所以我打算重启写博客的习惯了。

有这个想法的缘由也是有诸多事情的，从去年组建工作室以来，有过大强度的工作，有过空虚犹豫的时刻。随着工作室渐渐稳定下来，合理规划自己的生活成为了第一要务。写博客也是生活规划的一小部分，锻炼身体、学习编程、阅读同样都是生活规划的内容。这是缘由之一。昨天因为听了一期博客，突然对村上春树感兴趣了，便把他的一些文章拿来读读。《当我谈跑步时，我谈些什么》中的村上先生，每天早上5点钟起床跑步，中午写作，晚上独自听音乐。生活在偏远小城市的村上先生井井有条的、安静的、创造出名满世界的文字，这真是羡煞人也。这是缘由之二。由于写程序的原因，今年算是渐渐用到 Github 了。原本它只是一款代码托管商，但由于它的社交型让它变成了程序员的社交网站。更因为它本身具备的去中心化、开源的特性，它更是有了更多具有想象力的用处了。博客、宪法托管、协同翻译，无限的可能都可以有用户去开发想象。这是缘由之三。

### 怎么写，以什么方式写？

与之前的博客不同，这次的博客写作我决定用碎片整理的方式。在 Github 的公共仓库中新建了一个「notes」的仓库，这里会记录平常写的一些临时片段文章。如同代码管理一样，每次文章更新都是一次 commit，文章碎片的合并，版本的更新都是写博客的一部分。一次次 merg 后，正式版本的发布便是正式博客的发布。这正是 git 式的写作吧。

### 怎么才能做一个有趣的人？

当我订下计划和目标后，我怀疑这样有规律的生活是否是有趣的生活。可我反问自己，单调的生活可能不是有趣的生活。给自己多些尝试，多些体验吧。我做某一件事是因为我喜欢它，并不去期望它有什么作用。当生活丰富多彩时，你自会发现他们的意义了。
